# Copy All Errors（项目内开发插件）

此目录来自 [smartmita/godot-copy-all-errors](https://github.com/smartmita/godot-copy-all-errors)，固定于提交 `d8dd5bb29bba68d709435889c9c65c94f6fe7e7a`，许可证为 MIT，版权归 ShardSpace 所有。许可证原文见本目录 `LICENSE`。

用途：在 Godot 编辑器的“调试器 → 错误”面板加入“复制全部”按钮，便于复制错误和警告信息。它只在编辑器中运行，不属于游戏运行时功能。

启用：打开“项目 → 项目设置 → 插件”，将 **Copy All Errors** 设置为启用。插件依赖 Godot 编辑器内部控件结构；升级 Godot 后若按钮消失，应先检查上游兼容性。
